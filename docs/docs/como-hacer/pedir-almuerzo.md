# Pedir Almuerzos del Casino

Desde el portal puede seleccionar el **menú del casino** para uno o más alumnos con anticipación.

---

## Requisitos previos

!!! warning "Antes de pedir"
    - Su cuenta debe tener **saldo disponible** (ver [Abonar Saldo](abonar-saldo.md)).
    - El pedido debe realizarse **antes del horario límite** establecido por el casino.

---

## Cómo realizar un pedido

1. Inicie sesión y acceda al panel principal.
2. Haga clic en **Menú Casino** en el menú de navegación.
3. Se mostrará el listado de alumnos registrados en su cuenta.
4. Para cada alumno que desee pedir:
    - Seleccione la **fecha** del almuerzo (puede ser el día de hoy o días futuros disponibles).
    - Elija el **menú** del día.
5. Revise el resumen del pedido.
6. Haga clic en **Confirmar Pedido**.

---

## Horarios de pedido

| Tipo | Horario límite |
|---|---|
| **Pedido normal** | Hasta las 10:00 hrs del día del almuerzo. |
| **Pedido rezagado** | Hasta las 14:00 hrs (menú rezagados, disponible según configuración). |

!!! info "Pedidos fuera de horario"
    Si ya pasó la hora límite para el día de hoy, el sistema habilitará automáticamente la selección para el **día siguiente**.

---

## Estados de un almuerzo

| Estado | Descripción |
|---|---|
| **Pendiente** | El pedido fue recibido, el almuerzo aún no ha sido entregado. |
| **Entregado** | El almuerzo fue retirado por el alumno en el casino. |
| **Cancelado** | El pedido fue anulado y el monto no se descontó del saldo. |

---

## Pedidos para múltiples alumnos

Si tiene más de un alumno registrado, puede seleccionar el menú para cada uno de forma independiente en el mismo formulario. El sistema descontará el costo de cada almuerzo del saldo de su cuenta.

---

## Preguntas frecuentes

??? question "¿Puedo cancelar un pedido ya realizado?"
    Sí, siempre que el almuerzo no haya sido entregado. Contacte al personal del casino para gestionar la cancelación.

??? question "¿Qué pasa si no hay saldo suficiente para el pedido?"
    El sistema no permite confirmar un pedido si el saldo es insuficiente para cubrir el total. Deberá [abonar saldo](abonar-saldo.md) primero.

??? question "¿Puedo pedir para varios días a la vez?"
    Sí, puede seleccionar distintas fechas para cada alumno en el mismo formulario de pedido.

??? question "¿Qué incluye el menú rezagados?"
    Es una alternativa de menú disponible para pedidos tardíos. Su precio y composición son definidos por el casino. Consulte con el personal para más detalles.

??? question "¿El sistema me avisa cuando el alumno retira el almuerzo?"
    Si tiene activada la opción **Notificación de compra** en sus ajustes, recibirá un correo cuando se registre la entrega del almuerzo.
