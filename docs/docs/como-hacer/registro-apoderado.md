# Registrarse como Apoderado

El proceso de registro como apoderado se completa en **3 pasos** a través del asistente de configuración del sistema. Al finalizar, el sistema enviará una notificación al equipo del casino.

---

## Paso 1 — Datos del Apoderado

1. Inicie sesión con su cuenta de correo electrónico en [sabormirandiano.cl](https://sabormirandiano.cl).
2. Al acceder por primera vez, el sistema lo redirigirá automáticamente al asistente de registro.
3. Complete el formulario con:

| Campo | Descripción |
|---|---|
| **Nombre completo** | Nombre del apoderado tal como aparecerá en el sistema. |
| **Correo electrónico** | Se muestra el correo con que inició sesión (no editable aquí). |
| **Teléfono** | Número de teléfono de contacto (pre-completado desde su perfil). |
| **Alumnos a registrar** | Cantidad de alumnos que desea inscribir (1, 2, 3 o 4+). |

4. Haga clic en **Continuar**.

!!! info "Roles"
    Una vez completado este paso, el sistema asigna automáticamente el rol de **Apoderado** a su cuenta.

---

## Paso 2 — Datos de los Alumnos

Por cada alumno indicado en el paso anterior, complete:

| Campo | Descripción |
|---|---|
| **Nombre** | Nombre completo del alumno. |
| **Curso** | Curso al que pertenece (seleccione de la lista). |
| **Edad** | Edad del alumno. |
| **Restricciones alimentarias** | Alergias o restricciones de dieta (opcional, puede seleccionar varias). |

Haga clic en **Guardar y Continuar** al completar los datos de todos los alumnos.

---

## Paso 3 — Preferencias y Configuración

Configure las opciones de notificación y límites de gasto:

| Opción | Descripción |
|---|---|
| **Comprobante de transferencia** | Recibe un comprobante por correo al abonar saldo. |
| **Notificación de compra** | Recibe un correo cada vez que se registra un almuerzo (activado por defecto). |
| **Informe semanal** | Recibe un resumen de consumo cada viernes por correo. |
| **Tag compartido** | Permite que sus alumnos compartan el tag con otros alumnos. |
| **Correo alternativo** | Dirección de correo adicional que recibirá copias de todas las notificaciones. |
| **Límite de notificación de saldo** | Si el saldo baja de este monto, recibirá una alerta (por defecto: $1.500). |
| **Máximo diario** | Límite de gasto por alumno en un día (en pesos, por defecto: $2.000). |
| **Máximo semanal** | Límite de gasto por alumno en una semana (en pesos, por defecto: $8.000). |

!!! warning "Límites de gasto"
    Los límites de gasto aplican a **cada alumno** por separado. Si un alumno supera el límite, el sistema impedirá nuevos pedidos hasta el siguiente período.

Haga clic en **Guardar y Finalizar** para completar el registro.

---

## Confirmación

Al finalizar el asistente, el sistema:

- Envía una notificación al equipo del casino para activar la cuenta.
- Muestra la pantalla de confirmación y activa el acceso al panel principal (dashboard).

!!! success "¡Registro completado!"
    A partir de este momento puede abonar saldo y realizar pedidos de almuerzos para sus alumnos.

---

## Preguntas frecuentes de este proceso

??? question "¿Puedo agregar más alumnos después de completar el registro?"
    Sí. Contacte al administrador del sistema para que agregue alumnos adicionales a su cuenta.

??? question "¿Qué hago si me equivoqué en un dato durante el registro?"
    Puede modificar sus preferencias de notificación y límites de gasto desde la sección **Ajustes** una vez finalizado el registro. Para cambios en los datos de los alumnos, contacte al administrador del casino.

??? question "¿Por qué no puedo seleccionar el curso de mi hijo?"
    La lista de cursos es configurada por el administrador del casino. Si el curso no aparece, contacte al personal para que lo agreguen.

??? question "¿Para qué sirve el 'límite de notificación de saldo'?"
    Cuando el saldo disponible en su cuenta baja del monto configurado, el sistema le enviará una notificación recordándole que debe recargar saldo para continuar haciendo pedidos.
